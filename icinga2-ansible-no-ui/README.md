Role Name
========

Ansible role to install Icinga2 Headless alongside Plugins

Requirements
------------

EPEL

Example Playbook
-------------------------

```yaml
---
- hosts: MonitoringServers
  roles:

   - role: icinga2-ansible-no-ui
     check_commands:
       check_nrpe: |
          "-H", "$address$",
              "-c", "$remote_nrpe_command$",
     tags: icinga2-no-ui
```
